# Ionic Test app

## How to run the app

Execute these command lines:

1. Install node.js 

2. Install Ionic CLI
```
npm install -g @ionic/cli
```

3. Go in the ionic-test/app-test folder and execute the following line: 
```
ionic serve
```

A new window opens in your browser. The application will then be running there.
